# FlashBang

A chill flashcard game with satisfying card-flip animations. No backend, just vibes.

## Features

- **Card flip** — Click or tap to reveal the answer with a smooth 3D animation
- **Display modes** — Show word first, definition first, or a random mix
- **Shuffled play** — Cards are randomized each session with no repeats
- **Difficulty levels** — Filter cards by level (1-5)
- **Keyboard navigation** — Arrow keys for prev/next, Space/Enter to flip
- **Mobile-friendly** — Responsive design that works on any screen

## Decks

- **SAT / GRE Vocabulary** — 200 essential words (in progress)

## Development

No build step required. Open `index.html` in your browser or run:

```bash
npx serve .
```
